# Mi primera entrada al blog

¡Bienvenidos a mi blog! Este es el inicio de una nueva aventura donde compartiré mis pensamientos, proyectos y aprendizajes.

## ¿Por qué empezar un blog?

Siempre he querido tener un espacio para compartir lo que aprendo y las experiencias que voy viviendo en el mundo del desarrollo web y la tecnología.

## Tecnologías

Este blog está construido con:
- **Astro**: Un framework moderno para construir sitios web rápidos
- **Tailwind CSS**: Para los estilos
- **Markdown**: Para escribir contenido de forma sencilla

¡Espero que disfruten el contenido!
